# パーソナリティ診断システム (HPCS)

## 概要

HPCS は、ビッグファイブ理論に基づいたパーソナリティ診断システムです。74 の質問に回答することで、以下の 5 つの性格特性を測定します：

- 神経症傾向 (Neuroticism)
- 外向性 (Extraversion)
- 誠実性 (Conscientiousness)
- 協調性 (Agreeableness)
- 開放性 (Openness)
